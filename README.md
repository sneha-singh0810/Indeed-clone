# 💼 Indeed Clone

A full-stack web application that replicates the core functionality of the popular job search platform **Indeed**. This project allows users to search for jobs, and enables companies to post job listings.

---

## 💻 Technologies Used

### Frontend
- ⚛️ **React.js**
- 🎨 **Material-UI (MUI)**

### Backend
- ☕ **Spring Boot**
- 🛠 **Hibernate**

### Database
- 🍃 **MongoDB**
